## Remaining Limitations

1. **ZH inline tables**: Chinese generator still has inline table arrays for most sections. It now loads G but doesn't consume it for tables yet. This is a large refactor best done as a dedicated pass.
2. **Buff effect metadata**: Accessory and buff UpdateAccessory logic remains too complex for regex extraction. Effect descriptions are curated in guide_content.json.
3. **Shop prices**: Item prices are not in shops.json because they're defined in each item's SetDefaults. Cross-referencing would require joining shops.json with mechanical item data.
4. **Schema coverage**: Only wands.schema.json and bosses.schema.json exist. Enemies, accessories, shops need schemas.
5. **Release manifest**: Only generated by CI (not local pipeline). Could add a local generation step.
